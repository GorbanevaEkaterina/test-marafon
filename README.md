# Описание проекта

В данном проекте была выполнена семантическая вёрстка главной страницы с последующей адаптацией под два разрешения экрана: 1024px и 320px. Применялся подход "десктоп-фёрст" с разрешением от 1920px.

## Особенности реализации

- **Семантическая вёрстка**: Все HTML-элементы структурированы в соответствии с их назначением, что облегчает восприятие кода и его поддержку.
- **Адаптивность**: Страница адаптируется под разные размеры экранов:
  - Для экранов 1024px (планшеты) и 320px (мобильные устройства) предусмотрены соответствующие стили.
- **Мобильная версия**: В мобильной версии шапка приложения перестраивается в "бургер-меню":
  - При клике на логотип в шапке появляется анимированное меню для смены темы страницы.
- **Учет пользовательских предпочтений**: При смене темы страницы учитываются пользовательские предпочтения, сохраняя выбранную тему даже при перезагрузке страницы.
- **Методология БЭМ**: В названии классов используется методология БЭМ (Блок, Элемент, Модификатор), что обеспечивает ясную структуру и читаемость кода.
- **Код написан на JavaScript**: Логика взаимодействия реализована с использованием Vanilla JS без дополнительных библиотек.

## Планы на будущее

В будущем планируется:
- Разбить проект на модули для более легкой поддержки и масштабируемости.
- Переписать код на TypeScript, что обеспечит наличие строгой типизации и повысит надёжность приложения.

Таким образом, проект сочетает современные практики веб-разработки и направлен на создание высококачественного, адаптивного пользовательского интерфейса, учитывая предпочтения пользователей для оптимального взаимодействия.
